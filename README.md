## 🚀 Getting Started

First of all, you need to have node and yarn installed on your machine.
Then you can clone the repository.

`git clone https://github.com/LauraCastilho/LoginPage.git`

Now, start the web application
1. cd login
2. yarn install
3. yarn start

## 💻 Technologies
- ReactJS
- CSS
- HTML
- Visual Studio Code
